## 📖 ZipList 核心概念

### 内存布局

```
+----------+----------+----------+----------+-----+
| zlbytes  | zltail   | zllen    | entries  | end |
| (4 byte) | (4 byte) | (4 byte) | (变长)   | (1) |
+----------+----------+----------+----------+-----+
```

### Entry节点结构

```
+----------+----------+----------+
| prevlen  | encoding | content  |
| (1 or 5) | (1)      | (变长)   |
+----------+----------+----------+
```

### Prevlen编码规则

- **小节点** (前节点 < 254字节): prevlen用1字节存储
- **大节点** (前节点 ≥ 254字节): prevlen用5字节存储（第一字节为0xFE标记）

### 连锁更新

当插入大节点时，后续节点的prevlen可能需要从1字节扩展到5字节，导致该节点也变成大节点，进而触发下一个节点的prevlen扩展，形成连锁反应。
